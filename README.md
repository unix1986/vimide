vimide
======

an auto installer for vimrc and some vim plugins

Components:

    Ctags                 // ctags -R
    Cscope                // ctrl+] and ctrl+T .etc
    a.vim                 // switch .h and .cpp
    bufexplorer.vim
    taglist.vim
    vcscommand.vim        // use VCSVimDiff VCSVimBlame .etc
    winfileexplorer.vim
    winmanager.vim
Vimrc:
    a basic configure
